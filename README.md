# Online Course Website (COMP 3820SEF Project)

## Project Description
This is a comprehensive Online Course Platform developed for the **COMP 3820SEF** course. The project aims to provide a secure and user-friendly environment for students to browse courses and for instructors to manage educational content.

Key technologies used:
- **Backend:** Java, Spring Boot, Spring Security
- **Database:** JPA / Hibernate
- **Build Tool:** Gradle
- **Frontend:** JSP, Bootstrap

## 🛠️ How to Run
1. Clone the repository.
2. Open in IntelliJ IDEA.
3. Wait for Gradle to sync dependencies.
4. Run `Application.java`.
5. Access the website at `http://localhost:8080`.
